# cache22 — cachy family base packages (cachy-kde and cachy-server share these).
#
# These variants are built on the cachyos/cachyos-v3 base image and pull
# from CachyOS's repos. Per-kernel modules (nvidia-open, zfs, r8125) ship
# pre-built so installs don't need DKMS at runtime.
#
# To remove the cachy family entirely (e.g. CachyOS redistribution
# concerns): delete packages/cachy-*.txt, scripts/inject-custom-repos-cachy.sh,
# the cachy-* matrix entries in .github/workflows/build-image.yml, and
# the cachy-* entries in variants.json. The arch family is independent.

# ─── bootc + immutable infrastructure ─────────────────────
# bootc + bootupd both come from cmspam/bootc-v3 (built for x86-64-v3

# bootloader (shim/grub2) for atomic updates on `bootc upgrade`.
