# cache22 — arch family base packages (arch-kde and arch-server share these).
#
# These variants are built on the archlinux/archlinux base image and
# pull from ALHP (x86-64-v3 optimized rebuilds of stock Arch), stock
# Arch (fallback for anything not in ALHP), LizardByte's pacman repo
# (sunshine), the cmspam/* repos for bootc, qemu, virglrenderer,
# gamescope, and the in-image cache22-aur repo (AUR packages we build
# ourselves with v3 flags during the image build). DKMS-built kernel
# modules (nvidia-open, r8125, broadcom-wl, xone) compile at image-
# build time against linux-headers.
#
# Independent of the cachy family — designed to keep working if the
# cachy-* files get deleted entirely.
