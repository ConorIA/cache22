# Getting Started

This section covers installing cache22 from the live ISO and completing the one-time Secure Boot key enrollment on first boot.

The order to follow is:

1. [Installation](./installation/). Boot the ISO, run `cache22-install`, pick variant and disk options.
2. [First-Boot Secure Boot Setup](./secure-boot-first-boot/). Put firmware in setup mode before the first boot of the installed system so cache22 can enroll its keys.
3. [Variants](./variants/). What each variant ships and how to pick one.

After first boot completes, see [Updates and Reboots](../updates-and-reboots/) for the day-to-day update workflow.
